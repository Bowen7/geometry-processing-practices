# Geometry Processing Practices

### Tech Stack

#### TS, for rendering
- react
- three.js
- react-three-fiber

#### Rust, for processing geometry
- faer

### Prerequisites
- Node.js
- Rust
- [wasm-pack](https://github.com/drager/wasm-pack)
- [watchexec](https://github.com/watchexec/watchexec)

### Quick Start
```
pnpm install
pnpm dev
```
